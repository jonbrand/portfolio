## 🤖 Installation 

1. Clone the repo
   ```sh
   git clone https://github.com/jonbrand/my-portfolio.git
   ```
2. Install YARN packages
   ```sh
   yarn
   ```
3. run the development server
   ```sh
   yarn start
   ```
